## Project Overview

The To-Do List Application is a simple task management project that allows users to add, view, update, and delete daily tasks. It helps users organize their work efficiently and keep track of pending and completed tasks.

This project focuses on understanding basic CRUD operations and building a clean, user-friendly interface.

## Objectives

Manage daily tasks effectively

Learn basic application logic and state handling

Practice CRUD operations (Create, Read, Update, Delete)

Improve front-end and/or backend development skills

## Features

Add new tasks

Mark tasks as completed

Edit existing tasks

Delete tasks

Simple and user-friendly interface
